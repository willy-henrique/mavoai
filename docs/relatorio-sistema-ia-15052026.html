O sistema possui três modos de resposta automática:
(1) Resposta Rápida — para perguntas diretas e problemas simples, responde imediatamente sem investigação.
(2) Procedimento Passo a Passo — quando o cliente pede "como fazer", vai direto ao tutorial sem perguntas adicionais.
(3) Investigação Completa — para problemas complexos, coleta informações, analisa causa raiz e propõe solução detalhada.

Observação sobre categorização:
Os atendimentos classificados como "Não categorizado" (52 casos) referem-se a conversas vagas, contatos fora do horário ou tickets iniciados sem descrição do problema. A IA já é treinada para lidar com esses casos solicitando informações ao cliente.

Os 5 PDFs importados cobrem os tópicos mais frequentes: Software (ERP), Hardware (impressoras) e Fiscal — que juntos representam mais de 70% dos atendimentos.
